# 🃏 Battle Cards

A simple turn-based card game developed in Unity.

## 🎮 Features
- Turn-based combat system
- Random card draw (infinite deck)
- Attack, Defense, Heal, Special cards
- Win/Lose system with UI panels
- Custom cursor and sound effects

## 🕹️ How to Play
- Draw cards each turn
- Play your card
- Defeat the enemy before they defeat you

## 🛠️ Built With
- Unity
- C#
